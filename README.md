# 🌱 Soil Moisture Monitoring System using ESP8266

## 📌 Project Overview

The Soil Moisture Monitoring System is an IoT-based project that measures soil moisture levels using a soil moisture sensor and ESP8266 NodeMCU. The collected data is displayed on a web dashboard, allowing users to monitor soil conditions in real time and determine whether the soil is wet or dry.

## 🚀 Features

- Real-Time Soil Moisture Monitoring
- Web-Based Dashboard
- Soil Dry/Wet Status Detection
- ESP8266 Wi-Fi Connectivity
- Automatic Data Updates
- User-Friendly Interface

## 🛠️ Hardware Components

- ESP8266 NodeMCU
- Soil Moisture Sensor
- Jumper Wires
- USB Cable
- Wi-Fi Network

## 🔌 Circuit Connections

| Component | ESP8266 Pin |
|-----------|------------|
| Soil Moisture Sensor (AO) | A0 |
| VCC | 3.3V |
| GND | GND |

## 💻 Software Used

- Arduino IDE
- ESP8266 Board Package
- HTML
- CSS
- JavaScript
